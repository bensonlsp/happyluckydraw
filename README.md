# 幸運大抽獎機 Lucky Draw Machine

一個仿六合彩攪珠機風格的互動式網頁抽獎程式，適合公司年會、活動現場投影展示使用。

## 功能

- **49 個彩色波球**的物理模擬攪珠動畫（紅、藍、綠三色）
- **姓名滾輪**效果揭曉幸運兒
- **煙火、彩帶、愛心特效**
- **勝利號角**音效（仿 Mario Course Clear）
- **攪珠音樂**（BPM 150 循環，按下開始後自動播放）
- **幸運兒紀錄**：不重複抽獎，同名參與者亦可正確處理
- **自訂標題**：可更改主標題與副標題
- **LocalStorage 持久化**：重新整理頁面後名單與紀錄保留
- **手機響應式佈局**：側邊欄以抽屜滑入方式呈現
- **視窗自適應**：機器隨視窗高度縮放；視窗過矮時改為捲動模式

## 使用方式

1. 直接在瀏覽器開啟 `index.html`（無需伺服器）
2. 在左側「抽獎名單」欄位貼上參與者名字，**每行一個**
3. 按 **✓ 確認名單** 更新剩餘人數
4. 點選右上角或側邊欄音效按鈕**開啟音效**
5. 按紅色 **開始攪珠** 按鈕，等待 3 秒後揭曉幸運兒
6. 幸運兒彈出後，按**任意處**關閉，繼續下一輪
7. 已中獎者不會再被抽中（支援同名參與者）
8. 左側 **⚙️ 自訂標題** 可更改活動名稱
9. 按 **清除紀錄重置** 可重新開始全部抽獎

## 技術細節

| 項目 | 說明 |
|------|------|
| 技術棧 | 純 HTML / CSS / JavaScript，無需任何框架或建置工具 |
| UI 框架 | Tailwind CSS（CDN） |
| 字型 | Noto Sans HK（Google Fonts） |
| 音效 | Web Audio API 合成，無外部音訊檔案 |
| 動畫 | SVG 物理模擬 + Canvas 星空 / 煙火 |
| 持久化 | localStorage |

## 已知限制

- 需要現代瀏覽器（Chrome / Safari / Firefox 最新版）
- 音效需要使用者互動才能啟用（瀏覽器安全限制）
- 無網絡環境下 Tailwind CSS 與 Google Fonts 無法載入（功能正常，外觀降級）
- 重新整理後波球視覺狀態重置，但名單與幸運兒紀錄保留
